# image2image v0.3.0

Released: v0.3.0 — 2026-06-26

## Highlights

v0.3.0 is the largest release to date, adding a batch registration runner,
OME-ZARR support, modular application architecture, an Inno Setup Windows
installer, a full MkDocs documentation site, and dozens of UI and stability
improvements.

## New features

- **Registration runner.** A new batch-oriented registration runner dialog
  (`dialog_runner.py`) lets you queue and execute multiple registration tasks
  in sequence, with CLI integration.
- **OME-ZARR support.** The viewer can now read and display OME-ZARR images.
- **Modular application architecture.** The elastix, valis, and register
  applications have been refactored into more modular components with shared
  mixins (`_dialog_mixins.py`), improving maintainability and reuse.
- **Windows Inno Setup installer.** Added `package/windows-pyinstaller/installer.iss`
  and `installer.ps1` for generating a proper Windows setup executable.
- **Documentation site.** A full MkDocs-based documentation site has been
  added under `docs/`, with per-application guides (convert, crop, elastix,
  fusion, merge, register, valis, viewer) and screenshots.
- **CITATION.cff.** Added a citation file for academic use.
- **New notebooks.** Added `2025_12_19_multiscene_czi.ipynb` for multi-scene
  CZI handling and `2026_01_14_join_czi.ipynb` for joining CZI files.

## Improvements

- **Key handling refactor.** Centralised keyboard event handling in
  `_dialog_base.py`, reducing duplication across dialogs.
- **Qt bug fix.** Fixed an annoying Qt styling issue by removing a
  problematic stylesheet and improving widget initialisation.
- **Output format validation.** Added extra checks for supported output
  formats in the convert, crop, and runner dialogs.
- **UI cleanup.** Harmonised widget layouts, labels, and spacing across all
  dialogs.
- **Logo update.** Refreshed application icons (`.icns`, `.ico`, `.png`) with
  smaller, cleaner assets.
- **Pandas migration.** Updated to `pandas >= 1` API compatibility.
- **Path handling.** Fixed path resolution issues in the WSI and list
  subsystems.
- **macOS build script.** Improved `package/macos-pyinstaller/build.sh`
  robustness.

## Bug fixes

- Fixed edge cases in elastix 3D key handling and registration fiducial
  selection.
- Fixed several path-joining and file-location regressions after the
  module refactor.
- General stability fixes across the viewer, crop, valis, and merge dialogs.

## Known issues

- None reported.

## Download notes

- macOS: open the downloaded `.dmg` then drag image2image into Applications if prompted.
- Windows: open the downloaded `.zip` archive and run the `image2image.exe`.
- Linux: extract the downloaded archive and run the included `image2image` executable.
