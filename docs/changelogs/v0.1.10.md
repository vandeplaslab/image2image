# image2image v0.1.10

Released: v0.1.10 — 2025-04-29

## Highlights

This release improves the image registration workflow with a redesigned layout
and fixes several bugs in packaging and dataset handling. A new CLI test suite
has been added to help catch regressions early.

## New features

- Added initial CLI test suite (`tests/test_cli.py`).

## Improvements

- Redesigned layout for the registration dialog and channel/sample selectors.
- README updated with clearer project information.

## Bug fixes

- Added missing scikit-learn packaging hook to prevent import errors in
  bundled builds.
- Fixed dataset and list dialog display issues.
- Fixed edge cases in image preprocessing and file location dialogs.

## Known issues

- None reported.

## Download notes

- macOS: open the downloaded `.dmg` then drag image2image into Applications if prompted.
- Windows: open the downloaded `.zip` archive and run the `image2image.exe`.
- Linux: extract the downloaded archive and run the included `image2image` executable.
