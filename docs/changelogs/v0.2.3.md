# image2image v0.2.3

Released: v0.2.3 — 2025-12-15

## Highlights

This release introduces a major refactor of the crop/mask subsystem, a Windows
installer script, a significantly expanded test suite, and numerous fixes and
CI improvements accumulated across the v0.2.x series.

## New features

- **Crop/mask refactor.** The cropping and masking pipeline has been reworked
  with a dedicated `utils/crop.py` module and an overhauled crop dialog UI
  for a smoother image-cropping experience.
- **Windows installer script.** Added `scripts/install_win.ps1` to simplify
  installation on Windows.
- **Expanded test suite.** Added new test modules for registration
  (`test_register.py`), Windows-specific behaviour (`test_windows.py`),
  tutorials (`test_tutorials.py`), and the viewer (`test_viewer.py`).
- **New notebook.** Added `2025_11_19_fix_czi.ipynb` in the scratch directory
  for repairing CZI files.

## Improvements

- Fixed filtering logic in list-based dialogs (channel and sample selectors).
- Updated macOS PyInstaller specification and packaging hooks (napari,
  image2image runtime hooks).
- Network module (`_network.py`) improvements.
- Multiple `pyproject.toml` refinements.

## Bug fixes

- Assorted fixes across the v0.2.0, v0.2.1, and v0.2.2 patch releases,
  covering UI edge cases and packaging stability.
- Fixed `__init__.py` and `utilities.py` regressions.

## Known issues

- None reported.

## Download notes

- macOS: open the downloaded `.dmg` then drag image2image into Applications if prompted.
- Windows: open the downloaded `.zip` archive and run the `image2image.exe`.
- Linux: extract the downloaded archive and run the included `image2image` executable.
